nginx-php8.2
==========
Build image
-----------

```bash
git clone https://github.com/andreal01/Docker-Nginx-PHP-MySQL.git
cd Docker-Nginx-PHP-MySQL

```

Support Laravel
-------------
```
Support Laravel > 10
```

Run on container
-------------
```bash
docker-compose up -d
```
or in new Docker
```bash
docker compose up -d
```
